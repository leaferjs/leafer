## Leafer

绚丽多彩的 HTML5 Canvas 2D 图形渲染引擎，可结合 AI 绘图、生成界面。能让你拥有瞬间创建 100 万个图形的超强能力，免费开源、易学易用、场景丰富。

Leafer 定义了图形渲染引擎的核心功能，提供超强的动力，可以瞬间创建 100 万个图形，与具体 UI、业务场景无关， 提供了性能优化和可扩展的空间，遵循 Leafer 接口之上开发的渲染器、布局器、UI 框架之间可以互通有无。
