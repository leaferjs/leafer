# leafer

一个可以流畅运行100万个图形的渲染引擎, 使用TypeScript编写, 可以用来开发高性能的跨平台2D图形图像应用。

## 性能

目前在性能方面，通过对比已完全超过Adobe Illustrator, Sketch、Figma等国外知名图形设计软件，达到10万个图形时他们已无法进行正常操作，而Leafer却仍然可以以60fps的流畅速度进行操作（chrome浏览器下运行）。

性能对比视频：https://www.bilibili.com/video/BV1PU4y1t7L8

## 体验性能

PC端演示地址：https://leaferjs.github.io/demo

## 功能亮点

- 支持WebGL和Canvas 2D
- 通过渲染插件可以支持小程序、Node.js、桌面、移动端等不同的渲染环境
- 通过自定义UI系统，可扩展SVG、DOM、游戏精灵等不同的UI体系
- 针对4K以上的大屏，支持将内容渲染到多个拼合画布，提高超大画布的渲染性能
